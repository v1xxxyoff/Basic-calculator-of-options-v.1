# Basic Calculator of Options v.1

This primitive calculator is used for pricing options using a 10-period binomial interest rate model.

## Features
- Zero-coupon bond (ZCB) pricing
- Forward price calculation
- Futures price calculation
- Call option pricing

## Repository Contents
- `calc.py` - Main Python application
- `ivanov_'2lab_1(21)VAR.xlsx` - Excel file with calculation proofs and validation
- `calc.exe` - Standalone executable for users without Python/libraries
